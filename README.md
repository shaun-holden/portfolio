# DeShaun Holden — Portfolio

Personal portfolio showcasing full-stack projects built with modern web technologies.

**Live:** [deshaunholden.com](https://deshaunholden.com)

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Deployment:** Vercel
